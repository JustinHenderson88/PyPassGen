Usage Instructions :-

On Windows :-

1) Enter this on the command line :- PyPassGen.py 2) It will ask you to enter the number of charectars you want in the password. 3) After entering the desired number of charectars , press enter 4) You will see the generated password on the screen.

On Ubuntu :-

1) Enter this on the command line :- Python PyPassGen.py 2) It will ask you to enter the number of charectars you want in the password. 3) After entering the desired number of charectars , press enter 4) You will see the generated password on the screen.
